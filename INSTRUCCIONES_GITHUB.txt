PAQUETE DEFINITIVO PARA GITHUB PAGES

1. Crea o abre el repositorio de GitHub.
2. Sube directamente TODOS los archivos y carpetas contenidos en este ZIP a la raíz del repositorio.
3. En Settings > Pages, selecciona Deploy from a branch.
4. Elige la rama main y la carpeta /(root).
5. Guarda y espera a que GitHub publique el sitio.

IMPORTANTE
- index.html debe quedar en la raíz del repositorio.
- No subas la carpeta contenedora del ZIP; sube su contenido.
- Conserva las carpetas assets y documentos.
- La primera recarga tras reemplazar una versión anterior puede requerir Ctrl + F5.

Contenido verificado

- Ejercicios intermedios y variantes de decisión.

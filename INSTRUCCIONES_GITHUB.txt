PAQUETE AUTOCONTENIDO PARA GITHUB PAGES

1. Elimina del repositorio los archivos de la versión anterior, especialmente assets, app.js, styles.css, sw.js y manifest.webmanifest.
2. Sube index.html, la carpeta documentos y el archivo .nojekyll a la raíz del repositorio.
3. En Settings > Pages usa Deploy from a branch, main y /(root).
4. Espera la publicación y abre la URL del sitio.
5. Haz una recarga forzada con Ctrl+F5.

Esta versión integra dentro de index.html todos los estilos, la programación, el logotipo y las imágenes. Sólo el PDF permanece como archivo independiente en documentos/.
